# 皮梦云黑插件

基于皮梦云黑库的 AstrBot 插件，提供**云端黑名单管理**能力。查询、添加、删除黑名单，自动同步、群组保护和自动踢人一应俱全。

## 功能特性

### 黑名单管理
- **查询黑名单**：检查用户 / 群组是否在黑名单中，支持本地缓存 + 云端实时双模式
- **添加黑名单**：将用户或群组添加到云端黑库
- **删除黑名单**：从云端黑库移除用户或群组
- **自动同步**：定时同步云端黑库数据到本地缓存，支持增量同步

### 群组保护
- **黑群自动退群**：Bot 处于黑名单群组时自动退出
- **管理员进群退群**：黑名单用户为群主 / 管理员加入群聊时，Bot 自动退群（可开关）
- **自动踢人**：等级 ≥ 3 的黑名单用户在群内踢出（可开关）
- **消息拦截**：黑名单用户与 Bot 互动时拦截并提醒（可开关）

### 权限安全
- **管理员鉴权**：使用 AstrBot 内置管理员系统，无需单独配置 OP 列表
- **Token 鉴权**：敏感操作需要皮梦云 Token
- **限流保护**：API 查询频率限制，防止 IP 被封
- **同步锁**：防止并发同步冲突
- **同步回滚**：同步失败自动恢复旧数据

## 安装

将本插件放入 AstrBot 的 `addons/` 目录即可。

> **注意**：仅支持 **aiocqhttp 平台**（NapCat / OneBot 协议）

## 配置

在 AstrBot WebUI 管理面板中配置，或在配置文件中添加：

```yaml
pimeng_blacklist:
  bot_token: ""                              # 必填。皮梦云黑库 Bot Token
  api_base: "https://cloudblack-api.07210700.xyz"  # 可选，默认即可
  sync_interval: 300                         # 同步间隔（秒），60~3600，默认 300
  enable_auto_kick: true                     # 自动踢出等级≥3 的黑名单用户
  enable_quit_on_admin_join: true            # 黑名单管理员进群时 Bot 退群
  enable_message_intercept: true             # 黑名单用户与 Bot 互动时拦截提醒
  request_timeout: 10                        # API 请求超时（秒），1~30
```

### 配置项说明

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `bot_token` | string | — | **必填**。从皮梦云黑库获取的 Bot Token |
| `api_base` | string | 默认地址 | API 地址，一般无需修改 |
| `sync_interval` | int | 300 | 自动同步间隔（秒），范围 60~3600 |
| `enable_auto_kick` | bool | true | 自动踢出等级≥3 的黑名单用户 |
| `enable_quit_on_admin_join` | bool | true | 黑名单管理员进群时 Bot 退群 |
| `enable_message_intercept` | bool | true | 群聊中仅黑名单用户与 Bot 互动时拦截提醒 |
| `request_timeout` | int | 10 | API 请求超时（秒），范围 1~30 |

## 命令

### 基础命令（所有人可用）

```
/bl_help                        - 显示帮助信息
/bl_check                       - 同时检查自己（用户 + 群组）
/bl_check <ID>                  - 检查指定 ID（同时查用户和群组）
/bl_check <ID> user             - 检查指定用户
/bl_check <ID> group            - 检查指定群组
```

### 管理员命令

```
/bl_status                      - 查看插件状态与同步信息
/bl_sync                        - 手动强制同步云黑库
/bl_add <ID> <原因> [等级] [类型]  - 添加黑名单
/bl_remove <ID> [原因] [类型]     - 移除黑名单
/bl_list [页码]                  - 查看黑名单列表
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `ID` | QQ 号或群号，也支持 **@提及** 自动提取 |
| `原因` | 必填。添加/移除的原因文本 |
| `等级` | 1-轻微 / 2-一般 / 3-平台 / 4-严重（等级4需在管理面板操作） |
| `类型` | 指定操作类型（不指定时默认 **用户**） |

### 类型别名

`类型` 参数支持以下别名，可以混用：

| 含义 | 英文 | 简写 | 中文 |
|------|------|------|------|
| 用户 | `user` | `-u` | `用户` |
| 群组 | `group` | `-g` | `群组` / `群` |

示例：
```
/bl_add @某人 恶意刷屏 3          - @提及自动提取 ID，添加为违规等级 3
/bl_add 123456 广告 2 -g         - 将群组 123456 添加为等级 2
/bl_add 123456 测试 1 用户       - 中文别名添加用户
/bl_remove 123456 误加 群组      - 中文别名移除群组
/bl_check 123456 -g             - 短参数 -g 查询群组
```

## 等级说明

| 等级 | 名称 | 说明 | 自动操作 |
|------|------|------|----------|
| 1 | 轻微 | 轻度违规，如轻微骚扰 | 仅拦截提醒 |
| 2 | 一般 | 一般违规，如刷屏 | 仅拦截提醒 |
| 3 | 平台 | 平台级违规，如诈骗 | 拦截 + 自动踢出 |
| 4 | 严重 | 严重违规，需管理面板操作 | 拦截 + 需手动处理 |

## 拦截机制

```
黑名单用户发消息
  ├─ 私聊 + 唤醒 Bot → 每天提醒一次 + 显示添加者
  ├─ 群聊 + @Bot/指令 → 拦截提醒 + 显示添加者
  ├─ 群聊 + 等级≥3 → 自动踢出 + 显示添加者
  └─ 群聊 + 普通消息 → 不拦截（可通过配置启用全拦截）
```

> 关于**添加者**：踢出 / 拦截 / 私聊提醒消息中会显示该用户是由谁添加的（如 `添加者：云黑管理员：术无`），方便群友了解来源。

## 同步机制

- **定时同步**：按配置间隔自动同步（默认每 5 分钟）
- **查询前同步**：查询不存在的记录时自动增量同步
- **操作后同步**：添加 / 删除后立即触发同步
- **手动同步**：通过 `/bl_sync` 命令强制同步（忽略冷却时间）
- **回滚保护**：同步失败时自动恢复旧数据，不会丢失本地记录

## 注意事项

1. **等级 4** 的用户需在皮梦云黑库管理面板操作，无法通过命令添加
2. **Bot Token** 是必填项，未配置时插件以只读模式运行
3. **限流保护**：API 查询有限频，避免频繁调用导致 IP 被封
4. **自动踢人** 需要 Bot 为群管理员，否则踢人失败仅记录日志
5. **消息拦截** 仅对 Bot 互动时生效（@Bot 或触发指令），避免干扰正常群聊
6. 对拦截 / 踢人有疑问的用户，请通过 [皮梦云黑库](https://云黑.皮梦.wtf) 申诉
