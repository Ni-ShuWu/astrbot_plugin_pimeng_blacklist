# 皮梦云黑插件

基于皮梦云黑库的AstrBot插件，支持查询、添加、删除黑名单，自动同步和群组保护。

## 功能特性

- ✅ **查询功能**：检查用户/群组是否在黑名单中
- ✅ **添加黑名单**：将用户添加到云黑库
- ✅ **删除黑名单**：从云黑库移除用户
- ✅ **自动同步**：定时同步云黑库数据
- ✅ **群组保护**：Bot在黑名单群组时自动退出
- ✅ **自动踢人**：等级≥3的黑名单用户自动踢出
- ✅ **权限管理**：使用AstrBot内置管理员系统
- ✅ **缓存优化**：智能缓存未被拉黑的用户

## 安装配置

### 1. 获取Bot Token

访问皮梦云黑库管理面板获取Bot Token。

### 2. 配置插件

在AstrBot配置文件中添加以下配置：

```yaml
pimeng_blacklist:
  bot_token: "你的Bot Token"  # 必填
  api_base: "https://cloudblack-api.07210700.xyz"  # 可选，默认值
  sync_interval: 300  # 同步间隔（秒），默认300秒（5分钟）
  clean_cache_max: 10000  # 干净用户缓存上限
  enable_auto_kick: true  # 是否启用自动退群
  request_timeout: 10  # API请求超时（秒）
```

也可以在AstrBot管理面板配置

## 使用命令

### 基础命令

- `/bl_help` - 显示帮助信息
- `/bl_status` - 查看插件状态
- `/bl_sync` - 手动同步云黑库

### 查询功能

- `/bl_check` - 检查自己是否在黑名单中
- `/bl_check <QQ号>` - 检查指定用户是否在黑名单中
- `/bl_check <群号> group` - 检查指定群组是否在黑名单中

### 管理功能（仅管理员）

- `/bl_add <QQ号> <原因> [等级=1]` - 添加用户到黑名单
- `/bl_remove <QQ号> [原因]` - 从黑名单移除用户
- `/bl_list [页码=1]` - 查看黑名单列表

## 配置说明

### bot_token

**类型**：字符串
**必填**：是
**说明**：从皮梦云黑库获取的Bot Token，用于API认证

### api_base

**类型**：字符串
**默认值**：https://cloudblack-api.07210700.xyz
**说明**：皮梦云黑库API地址

### sync_interval

**类型**：整数
**默认值**：300（5分钟）
**范围**：60-3600秒
**说明**：自动同步云黑库的间隔时间

### clean_cache_max

**类型**：整数
**默认值**：10000
**说明**：缓存未被拉黑的用户数量上限，避免内存占用过大

### enable_auto_kick

**类型**：布尔值
**默认值**：true
**说明**：Bot在黑名单群组时自动退出，并自动踢出等级≥3的黑名单用户

### request_timeout

**类型**：整数
**默认值**：10
**范围**：1-30秒
**说明**：API请求的超时时间

## 等级说明

- **等级1**：轻微违规
- **等级2**：一般违规
- **等级3**：平台违规
- **等级4**：严重违规（需在管理面板操作）

## 同步机制

插件实现了完整的同步机制：

- **定时同步**：每5分钟自动同步一次
- **查询前同步**：确保查询时数据是最新的
- **操作后同步**：添加/删除后立即同步
- **手动同步**：通过`/bl_sync`命令随时同步

## 拦截机制

- **私聊拦截**：全部拦截黑名单用户消息
- **私聊提醒**：仅主动唤醒bot时每天提醒一次
- **群聊拦截**：拦截黑名单用户消息，不发送提醒

## 群组保护

- **Bot退群**：当Bot在黑名单群组时自动退出
- **自动踢人**：等级≥3的黑名单用户自动踢出（仅群聊）
- **踢人消息**：仅踢人成功时发送，失败时不发送

## 注意事项

1. 等级4的用户需要在皮梦云黑库管理面板操作
2. Bot退群功能只对等级≥3的群组生效
3. 自动踢人功能只对等级≥3的用户生效
4. 插件使用AstrBot内置管理员系统，无需单独配置OP列表
5. 建议定期使用`/bl_sync`命令手动同步以确保数据一致性
